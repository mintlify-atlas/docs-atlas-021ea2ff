## GET /api/admin/dashboard

Returns a full operational summary for the current day. This endpoint powers the main admin dashboard panel.

<Note>Requires a valid JWT Bearer token in the Authorization header.</Note>

### Request

No parameters or request body required.

```bash
curl --request GET \
  --url https://your-domain.com/api/admin/dashboard \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```

### Response

**200 OK**

<ResponseField name="success" type="boolean" required>
  Always `true` on success.
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="properties" defaultOpen>
    <ResponseField name="resumen" type="object" required>
      Aggregate summary metrics for today.
      <Expandable title="resumen properties" defaultOpen>
        <ResponseField name="resumen.reservasHoy" type="number" required>
          Total number of reservations for today.
        </ResponseField>
        <ResponseField name="resumen.clientesActivos" type="number" required>
          Total number of active clients in the system.
        </ResponseField>
        <ResponseField name="resumen.ventasDelDia" type="number" required>
          Total sales value for today (sum of `ventas.total`).
        </ResponseField>
        <ResponseField name="resumen.canchasDisponibles" type="number" required>
          Number of courts not yet booked for today.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="reservasHoyDetalle" type="array" required>
      List of today's reservations, each with client name, sede, sport, court number, and time slot.
    </ResponseField>
    <ResponseField name="sedes" type="array" required>
      Status of each location: total courts, occupied courts, and operating hours.
    </ResponseField>
    <ResponseField name="actividadReciente" type="array" required>
      Up to 10 most recent events across reservations, sales, and new clients.
      <Expandable title="activity item properties">
        <ResponseField name="tipo" type="string">
          Event type: `"reserva"`, `"venta"`, or `"cliente"`.
        </ResponseField>
        <ResponseField name="timestamp" type="string">
          ISO 8601 timestamp when the event occurred.
        </ResponseField>
        <ResponseField name="descripcion" type="string">
          Human-readable event description.
        </ResponseField>
        <ResponseField name="detalle" type="string">
          Additional event detail (e.g., time slot range for reservations, payment method for sales).
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Always `"Dashboard obtenido correctamente"`.
</ResponseField>

### Error responses

| Status | Condition |
|---|---|
| `401 Unauthorized` | Missing, expired, or invalid Bearer token. |
| `500 Internal Server Error` | Database query failed. |
