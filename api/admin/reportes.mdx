## GET /api/reportes

Returns aggregated reports for the specified report type and date range. Use this endpoint to generate sales, reservation, or inventory summaries for a given period and sede.

<Note>Requires a valid JWT Bearer token in the Authorization header.</Note>

### Query parameters

<ParamField query="tipo" type="string">
  The type of report to generate. Example value: `"ventas"`.
</ParamField>

<ParamField query="fecha_inicio" type="string">
  Start of the date range in `YYYY-MM-DD` format.
</ParamField>

<ParamField query="fecha_fin" type="string">
  End of the date range in `YYYY-MM-DD` format.
</ParamField>

<ParamField query="sede_id" type="number">
  Filter report data to a specific sede by ID.
</ParamField>

### Response

**200 OK**

<ResponseField name="success" type="boolean" required>
  Always `true` on success.
</ResponseField>

<ResponseField name="data" type="object" required>
  Aggregated report data. The shape of this object varies depending on the `tipo` requested.
</ResponseField>

### Error responses

| Status | Condition |
|---|---|
| `401 Unauthorized` | Missing, expired, or invalid Bearer token. |
| `500 Internal Server Error` | Database query failed (`DATABASE_ERROR`). |
