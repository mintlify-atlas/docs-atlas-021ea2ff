---
title: 'Torneos (admin)'
description: 'Manage tournaments: list, create, register participants, record results, and manage brackets.'
---

All tournament admin endpoints require authentication and the `torneos` permission. Employees without `all: true` only see tournaments for their assigned location.

<Note>Requires a valid JWT Bearer token in the Authorization header with `torneos` permission.</Note>

---

## GET /api/torneos

Returns tournaments for the authenticated employee's location. Administrators see all tournaments across all locations.

```bash
curl --request GET \
  --url https://your-domain.com/api/torneos \
  --header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```

**Response:**

```json
{
  "success": true,
  "data": [
    {
      "id_torneo": 1,
      "nombre": "Copa Verano 2026",
      "fecha_inicio": "2026-02-01",
      "fecha_fin": "2026-03-31",
      "deporte": "Pádel",
      "sede": "Sede Principal",
      "activo": true
    }
  ]
}
```

---

## GET /api/torneos/ranking

Returns the overall ranking across all tournaments. Requires `torneos` permission.

---

## POST /api/torneos

Creates a new tournament.

### Request body

<ParamField body="nombre" type="string" required>
  Tournament name.
</ParamField>

<ParamField body="id_sede" type="number" required>
  ID of the hosting location.
</ParamField>

<ParamField body="id_deporte" type="number" required>
  ID of the sport for this tournament.
</ParamField>

<ParamField body="fecha_inicio" type="string" required>
  Start date in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="fecha_fin" type="string" required>
  End date in `YYYY-MM-DD` format.
</ParamField>

---

## GET /api/torneos/:idTorneo/participantes-inscritos

Returns the list of participants registered for a tournament.

<ParamField path="idTorneo" type="number" required>
  Tournament ID.
</ParamField>

---

## POST /api/torneos/:idTorneo/participantes

Registers participants in a tournament.

<ParamField path="idTorneo" type="number" required>
  Tournament ID.
</ParamField>

---

## GET /api/torneos/:idTorneo/partidos

Returns all matches (bracket) for a tournament.

<ParamField path="idTorneo" type="number" required>
  Tournament ID.
</ParamField>

---

## GET /api/torneos/:idTorneo/ganador

Returns the winner of a completed tournament.

<ParamField path="idTorneo" type="number" required>
  Tournament ID.
</ParamField>

---

## POST /api/torneos/:idTorneo/cerrar-inscripcion

Closes registration and generates the tournament bracket (match schedule).

<ParamField path="idTorneo" type="number" required>
  Tournament ID.
</ParamField>

---

## POST /api/torneos/:idTorneo/cancelar

Cancels a tournament.

<ParamField path="idTorneo" type="number" required>
  Tournament ID.
</ParamField>

---

## POST /api/torneos/partidos/:idPartido/resultado

Records the result of a match, advancing the winning participant in the bracket.

<ParamField path="idPartido" type="number" required>
  Match ID.
</ParamField>

---

## Error responses

| Status | Condition |
|---|---|
| `401 Unauthorized` | Token is missing or employee is inactive. |
| `403 Forbidden` | Token expired/invalid, or missing `torneos` permission, or sede scope denied. |
| `500 Internal Server Error` | Database error. |
