## GET /api/cliente/sedes

Returns all active sedes (locations) in the TANCAT system. This endpoint is public and requires no authentication. Use it to populate location pickers in client-facing interfaces.

### Request

No parameters or request body required.

```bash
curl --request GET \
  --url https://your-domain.com/api/cliente/sedes
```

### Response

**200 OK**

<ResponseField name="success" type="boolean" required>
  Always `true` on success.
</ResponseField>

<ResponseField name="data" type="array" required>
  Array of active sede objects.

  <Expandable title="sede properties">
    <ResponseField name="id_sede" type="number" required>
      Unique identifier for the sede.
    </ResponseField>
    <ResponseField name="nombre" type="string" required>
      Display name of the sede.
    </ResponseField>
    <ResponseField name="direccion" type="string" required>
      Street address of the sede.
    </ResponseField>
    <ResponseField name="horario" type="string" required>
      Operating hours description for the sede.
    </ResponseField>
  </Expandable>
</ResponseField>
