# TANCAT System

TANCAT System is a comprehensive sports complex administration platform built for Complejo Deportivo TANCAT, operating since 1991 in Córdoba, Argentina. It manages court bookings, inventory, sales, tournaments, and employees across three locations: Main, Jacinto Ríos, and Rincón.

## Tech stack

| Layer | Technology |
|---|---|
| Backend | Node.js + Express.js |
| Frontend | Vanilla JavaScript (ES6+) + Vite |
| Database | Neon PostgreSQL (serverless) |
| Auth | JWT (8-hour expiry) |
| Deployment | Vercel (São Paulo region) |
| Security | Helmet, CORS, Rate Limiting, bcrypt |

## Project locations

TANCAT operates three courts complexes, each with independent inventory management:

- **Main** (`id_sede: 1`) — Primary location, full feature set
- **Jacinto Ríos** (`id_sede: 2`) — Dedicated inventory management
- **Rincón** (`id_sede: 3`) — Dedicated inventory management

<Note>
  Employee access is scoped by location — staff at Jacinto Ríos can only access inventory and operations for their sede unless they have the `superadmin` role.
</Note>
